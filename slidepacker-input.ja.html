スライドの URI 参照

現時点では suika.fam.cx, HTTP のみ使用可能です。

パッケージの媒体型
application/zip
作成

事前に許諾を得た人のみ利用できます。

少々時間が掛かります。

解説

HSML プレゼンテーション・スライドの本体とそこから参照されているスタイル・シート一式 (CSS, XBL 1, HTC) を zip 形式で1つにまとめます。

実際には対象となる文書を単純な文字列とみなして、属性やスタイル・シート内の URI 参照のようなものを探して適当に置き換えています。 つまり、あらゆる場合に正確に対応できるわけではありません。

アクセス間隔やパッケージ化のための負荷などの問題があるので、 現時点で使用できる対象を suika.fam.cx に限定しています。更に、予め登録された人しか利用できません。 利用を希望される方はわかばまでご連絡下さい。
